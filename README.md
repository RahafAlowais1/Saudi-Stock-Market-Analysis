# 📈 Saudi Stock Market Analysis (Tadawul)
Exploratory Data Analysis of the Saudi Stock Market using Python

## 📝 Project Description
This project performs a comprehensive Exploratory Data Analysis (EDA) on the Saudi Stock Market (Tadawul) dataset. The goal is to uncover market trends, analyze price fluctuations, and identify the performance of various companies and sectors through data visualization and statistical summaries.

## 📊 Key Insights & Results
Based on the analysis conducted in the notebook, several key findings were identified:

Top Performance Identification: Identified the top 15 companies based on their Mean Closing Price, providing insights into high-value stocks in the market.

Industry Analysis: Aggregated data by "Trading Name" to compare how different industries perform in terms of average stock pricing.

Price Distribution: Analyzed the range of stock prices (Open, Close, High, Low) to understand market volatility and stability.

Data Consistency: Successfully handled and cleaned historical trading records, ensuring that the mean calculations are not skewed by missing values.

Foundation for Prediction: The processed data and identified trends serve as a primary stage for building future Machine Learning models for stock price forecasting.

## 🛠 Tech Stack & Libraries
The project is built using the following Python libraries:

Pandas: For data manipulation, cleaning, and grouping.

NumPy: For numerical operations and statistical calculations.

Matplotlib & Seaborn: For creating professional-grade visualizations and charts.

## 📂 Data Source

Data is typically sourced from Kaggle (https://www.kaggle.com/datasets/salwaalzahrani/saudi-stock-exchange-tadawul) or directly from the Tadawul official website.
